Chi tiết bài học

Bài học này sẽ hướng dẫn các bạn cách thức hoạt động của mô hình Web truyền thống (SSR) so với Web Service / API. Hiểu sâu hơn về ASGI Server và thiết lập dự án FastAPI cơ bản.

Tài liệu tham khảo & Hướng dẫn thiết lập FastAPI

I. Giới thiệu

FastAPI là một Web framework hiện đại, hiệu năng cao để viết các API bằng Python. Nó được thiết kế dựa trên các tính năng gợi ý kiểu (type hints) tiêu chuẩn của Python 3.8+.

II. Các tính năng cốt lõi

  • Nhanh: Hiệu năng tương đương NodeJS và Go (nhờ Starlette và Pydantic).
  • Hạn chế lỗi: Giảm khoảng 40% lỗi do nhà phát triển gây ra.
  • Trực quan: Hỗ trợ auto-complete tuyệt vời trên các IDE soạn thảo.
  • Tài liệu tự động: Tự động sinh giao diện Swagger UI tại đường dẫn /docs.

VI. Tài liệu tham khảo

  • FastAPI Documentation: https://fastapi.tiangolo.com/
  • MDN Web Docs: Client-Server Overview.
  • ASGI Specification: https://asgi.readthedocs.io/
